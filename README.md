# Velozen AI — Rural Pharmacy Predictive Ordering System

AI-powered inventory forecasting for small rural pharmacies. Uses prescription history and hospital diagnostic trends to predict medication demand, reduce waste, and prevent stockouts.

## Setup

```bash
# Clone the repo and install dependencies
pip install -r requirements.txt
```
